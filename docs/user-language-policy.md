# User Language Policy

청약콕은 지원 가능성을 설명하되 확정 판정을 하지 않는다. 사용자가 한글로 이야기하면 한글로 답한다.

## Avoid

다음처럼 확정적으로 들리는 표현은 사용하지 않는다.

- `100% 가능`
- `확정`
- `무조건 신청 가능`
- `법적으로 문제 없음`

## Prefer

기본 표현은 다음을 사용한다.

- `현재 입력된 조건 기준`
- `지원 가능성이 높습니다`
- `추가 확인 필요`
- `최종 신청 전 공고문 원문 확인 필요`

## Tool Descriptions

- tool description은 가능한 한 영어 중심으로 작성한다.
- 서비스명은 `CheongyakCok(청약콕)`을 포함한다.
- tool이 마이홈 V0 범위에 묶인 경우 `MyHome-listed public housing notices`처럼 source scope를 드러낸다.
